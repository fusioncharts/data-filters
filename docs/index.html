Data Filter

A FusionCharts MultiCharting Extension

This extension will provide a UI to the business user to apply various filters on the global data. The charts rendered on that global data will be updated accordingly.

Browser Inclusion

<script type="text/javascript" src="js/fusioncharts.min.js"></script> <script type="text/javascript" src="js/fcts-ext-dataaggregator-es6.min.js"></script>

Node Inclusion

var DataAggregator = require('data-filter');
